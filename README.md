# Evaluator-ENG-HIN
This is the tool for evaluating machine translation text for English-Hindi which was done as internship project at CDAC,pune

Evaluator is used to check the accuracy of Machine Translation Systems for English-Hindi language, 5 evaluation metrics have been used BLEU, NIST, METEOR, NIST & GTM.

Technologies used : Python, Django Framework, Bootstrap Framework, HTML, CSS
